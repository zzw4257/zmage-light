# 4. Zmage 媒体光谱：兼容现在，拥抱未来

Zmage 的设计不止于经典的 JPEG 图片，而是要成为能够容纳所有现代视觉媒介的统一载体。

*   **核心图片格式**:
    *   **JPEG, PNG, GIF, WebP**: 全面支持，并进行优化处理。
    *   **HEIC (高效图像格式)**: 自动识别苹果设备的 HEIC 格式，并在服务端无缝转换为高质量的 JPEG 进行预览和处理，同时保留原始 HEIC 文件。
*   **专业摄影支持**:
    *   **RAW 格式**: 支持主流相机厂商的 RAW 文件 (如 `.CR2`, `.NEF`, `.ARW`, `.DNG`)。
    *   **RAW + JPEG 智能堆叠**: 自动识别并关联同一场景的 RAW 和 JPEG 文件，在视图中堆叠显示，默认展示 JPEG，用户可随时切换查看 RAW 或将其设为首选。
    *   **16位色深图像 (TIFF, PSD)**: (规划) 支持预览，并保留原始文件用于专业工作流。
*   **视频作为一等公民**:
    *   **广泛格式支持**: 支持 MP4, MOV, AVI, MKV 等主流视频格式。
    *   **智能封面提取**: 通过 `fluent-ffmpeg` 分析视频内容，自动截取最具代表性的关键帧作为封面。
    *   **视频预览**: 在网格视图中，鼠标悬停可静音预览视频片段。
    *   **服务端转码**: (规划) 对上传的视频进行标准化转码（如转为 H.264 MP4），以保证在所有设备上的流畅播放。
    *   **自适应码流**: (规划) 为长视频生成多种分辨率的版本，根据用户的网络状况自动切换，实现类似 YouTube 的播放体验。
*   **动态与沉浸式媒体**:
    *   **Live Photo (实况照片)**:
        *   **原生体验**: 使用 **LivePhotosKit JS**，在 Web 端完美还原苹果设备的 Live Photo 动态播放效果。
        *   **智能拆分与关联**: 上传时自动拆分为静态图像和短视频，并进行强关联。
    *   **全景/360°照片**:
        *   **自动识别**: 根据元数据自动识别为全景照片。
        *   **交互式查看器**: 提供专用的、可拖拽、可缩放的全景查看器。
*   **非经典与未来媒体**:
    *   **高动态范围图像 (HDR)**: (规划) 支持 HDR 照片的元数据识别和预览。
    *   **空间照片/视频 (Spatial Photos/Videos)**: (规划) 为苹果 Vision Pro 等设备拍摄的空间媒体提供存储支持，并探索 Web 端的 3D 预览方案。
    *   **隐写图片 (Steganography)**: (规划) 提供工具，允许用户将少量文本信息以不可见的方式嵌入图片中，作为"数字保险箱"。
