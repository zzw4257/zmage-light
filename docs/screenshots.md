# Zmage 系统截图

## 主页面 - 资产管理
- 液态玻璃 UI 风格
- 左侧导航栏包含：全部资产、图片、视频、文档、文件夹、相册、AI 建议、集合、分享
- 顶部搜索栏支持 AI 语义搜索
- 网格视图展示资产

## 相册页面
- 支持创建相册
- 展示相册列表
- 空状态提示

## 设置页面
- 系统概览：总资产、存储使用、最近上传、待审核建议
- 资产分布：按类型统计
- 后台任务：队列长度、等待中、运行中、失败
- 系统信息：版本、前端、后端、AI 引擎
- 快捷入口：下载预设、Portal 管理、后台任务

## UI 特点
- 液态玻璃效果 (backdrop-filter: blur)
- 圆角卡片设计
- 渐变色图标
- 响应式布局
- 支持亮色/暗色主题
